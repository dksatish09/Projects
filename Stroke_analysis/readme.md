# Stroke Analysis

## Objective
To compare people with and without stroke and find key factors.

## Dataset
Rows: 5110  
Columns: 12  
Source: Kaggle

## Tools Used
Jupyter Notebook, Pandas, NumPy, Matplotlib, Seaborn

## Visualizations
- Stroke vs HearDisease
- Stroke vs hypertension
- Stroke vs glucose_level
- Stroke vs Smoking
- Stroke vs BMI comparison

## Business Insights
Females have higher stroke count due to high hypertension, overweight, and glucose levels.
